# Type Descriptions

This document contains descriptions and explanations of the different objects seen in ReactDraw

## Point

```typescript
type Point = [number, number];
```

## RectBounds

```typescript
type RectBounds = {
  top: number;
  right: number;
  bottom: number;
  left: number;
};
```

## DrawingContainer

```typescript
type DrawingContainer = {
  div: HTMLDivElement;
  id: string;
  bounds: RectBounds;
};
```

## DrawingData

```typescript
type DrawingData = {
  coords: Point[];
  container: DrawingContainer;
  element: HTMLElement | SVGSVGElement | null;
  style: {
    lineWidth: number;
    zIndex: number;
  };
  toolId: string;
};
```

## OnUpdateContext

```typescript
type OnUpdateContext = {
  viewContainer: HTMLDivElement;
  previousPoint: Point;
  newPoint: Point;
  mode: SelectMode;
};
```

## SelectMode

```typescript
type SelectMode =
  | "drag"
  | "rotate"
  | "resize-nw"
  | "resize-ne"
  | "resize-se"
  | "resize-sw";
```
